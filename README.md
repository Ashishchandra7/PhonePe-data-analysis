# 📊 PhonePe Data Analysis Project

## 📌 Project Overview

This project analyzes **PhonePe digital payment data** to understand transaction trends across India.
The analysis focuses on **state-wise, district-wise, and yearly transaction patterns** using Python and data visualization tools.

The goal of this project is to extract meaningful insights from large datasets and build an interactive data analysis workflow.

## 📈 Sample Insights

* Digital transactions increased significantly after 2020.
* Southern states show higher digital payment adoption.
* Metro cities dominate in transaction volume.

## 🛠 How to Run the Project

1. Clone the repository

```
git clone https://github.com/Ashishchandra7/PhonePe-data-analysis.git
```

2. Install required libraries

```
pip install -r requirements.txt
```

3. Run the Streamlit application

```
streamlit run app.py
```
